🔬 Featured Projects

Forest Restoration Risk Scoring Model
- A Python-based tool to quantify environmental vulnerability across reforestation sites using rainfall, slope, and erosion metrics. Built to support risk-aware decision-making in conservation planning.

Carbon MRV Benchmarking
- A literature review and impact framework comparison exploring how monitoring, reporting, and verification methods vary across carbon programs (Verra, Gold Standard, REDD+, Plan Vivo). Synthesizes key insights to support methodological harmonization.

Computer Vision PoC — Operational Use Case Pipeline
- A proof-of-concept pipeline for object detection across simulated terminal camera feeds. Benchmarks edge vs. cloud deployment architectures on accuracy, latency, and cost, and produces a risk-scored implementation roadmap.

Curbside Capacity & Flow Assessment
- A GIS-informed capacity analysis of curbside zones across a simulated three-level terminal. Identifies peak-period constraints, accessibility compliance gaps, and delivers prioritized program improvement recommendations.

Fleet Lifecycle Cost Analysis Model
- A TCO model evaluating replacement vs. sustainment strategies across a simulated fleet of 22 assets. Models maintenance cost trajectories and reliability degradation to identify optimal renewal timing.

Website Funnel & Clickstream Analysis
- End-to-end funnel analysis identifying drop-off points and behavioural patterns across a simulated user journey. Built to surface actionable conversion insights for commercial decision-making.
